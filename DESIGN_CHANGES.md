# Изменения дизайна диалога подписки

## Что было изменено:

1. **Современный дизайн диалога**:
   - Добавлена скругленная форма (RoundedCornerShape 28.dp)
   - Увеличен размер диалога до 92% ширины экрана
   - Добавлена тень (tonalElevation = 8.dp)
   - Иконка у заголовка для визуальной привлекательности

2. **Иконки на кнопках**:
   - Кнопка "Да" (подписаться): иконка ✅
   - Кнопка "Нет" (отказ): иконка ❌  
   - Кнопка "Больше не спрашивать": иконка 🚫

3. **Новый порядок кнопок**:
   - Слева: "Нет" (с иконкой ❌)
   - Справа: "Да" (с иконкой ✅)
   - Снизу: "Больше не спрашивать" (с иконкой 🚫)

4. **Улучшенный UI**:
   - Круглая форма кнопок (12.dp)
   - Цветовые стили Material 3
   - Оптимизированные отступы

## Файлы, которые были изменены:
- `android-app/app/src/main/java/com/alertsua/app/map/AlertMapScreen.kt` - реализация нового дизайна

## Технические детали:
- Добавлены импорты для Material Icons

- Реализованы кастомные цвета и тени
